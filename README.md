## About
### Simple gui client for [chatroom-server](https://github.com/KH4St3H/chatroom-server) project

## How to run
1. Install required packages
  `pip install pycryptodome`

2. Run using python
  `python3 gui.py`

## Features
- [x] Login
- [x] Register
- [x] Public message
- [x] Private message
- [x] Public file
- [ ] Private file
- [x] See list of poeple in the chatroom
